# Logo

Placez votre logo ici : `logo.png` ou `logo.svg`

Pour utiliser votre logo dans l'application, modifiez `src/pages/Client.jsx` :

```jsx
// Ligne ~42, remplacez l'emoji par :
<img src="/logo.png" alt="GLesCrocs" style={{ width: '100px', height: 'auto', marginBottom: '10px' }} />
```
